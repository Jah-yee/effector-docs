01 — CHECK

Type-safe validation

42 standard types. Structural subtyping. Manifest + types + lint + audit in one pass.

$ effector check . ✓ Types CodeDiff → ReviewReport ✓ Lint 0 warnings ✓ Audit Score 5/5
02 — COMPILE

Cross-runtime output

One manifest, any runtime. MCP, OpenAI Agents, LangChain, or raw JSON IR.

$ effector compile . -t mcp { "name": "code-review", "inputSchema": { ... } }
03 — AUDIT

Declared permissions

Network, filesystem, subprocess, env access. Declared in manifest, verified against source.

$ effector inspect . network yes subprocess no filesystem read env-read GITHUB_TOKEN
